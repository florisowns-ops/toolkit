## 🚀 部署（免费）

### Vercel（推荐）
1. 注册 [vercel.com](https://vercel.com)（GitHub 登录）
2. 把本项目 push 到 GitHub
3. 在 Vercel 导入 GitHub 仓库，点 Deploy
4. 获得免费域名 `xxx.vercel.app`

### 自定义域名
在 Vercel 设置里绑定你自己的域名即可，免费 SSL。
